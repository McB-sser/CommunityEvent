# Test-Checkliste

- Plugin startet ohne Fehler.
- Event kann erstellt/gestartet/beendet werden.
- Teilnehmer koennen beitreten und werden korrekt erfasst.
- Event-Status, Timer und Broadcasts werden sichtbar aktualisiert.
- Belohnungen oder Abschlusslogik funktionieren wie erwartet.
- Keine Errors oder Warnings in der Konsole.
